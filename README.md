# 🔐 Advanced Wordlist Generator GUI

A powerful and customizable **Wordlist Generator Tool** designed for cybersecurity enthusiasts, students, and penetration testers. This tool allows users to generate targeted wordlists based on personal information such as name, date of birth, pet name, and more.

---

## 🚀 Features

* 🧠 **Smart Wordlist Generation**

  * Generates combinations using user-provided personal data
  * Useful for password analysis and security testing

* ➕ **Dynamic Input Fields**

  * Add or remove input fields as needed
  * Flexible data entry (no fixed limit)

* 🖥️ **User-Friendly GUI**

  * Clean and simple interface
  * Built using Python (Tkinter / CustomTkinter)

* 📊 **Live Count Display**

  * Shows number of generated words

* 💾 **Export Support**

  * Save generated wordlists to a file

---

## 📌 How It Works

1. User enters target-related information:

   * Name
   * Date of Birth
   * Pet Name
   * Favorite Words
   * Any custom input

2. User can:

   * ➕ Add more input fields
   * ➖ Remove unnecessary fields

3. Click **"Generate Wordlist"**

4. Tool creates combinations such as:

   * `name123`
   * `petname@dob`
   * `name_pet_2024`
   * and many more...

## 🧩 Usage

* Open the application
* Enter target information in available fields
* Use `+` button to add more fields
* Use `-` button to remove fields
* Click **Generate Wordlist**
* View results in the output area
* Save if needed
